Olympic Games Data Analysis

An exploratory data analysis of historical Olympic Games data using Python, Pandas, Plotly, and Streamlit.

The project explores long-term Olympic trends, including the number of sports, participating countries, and medals over time. It also includes an extended analysis focused on the Czech Republic and Slovakia, together with an interactive Streamlit dashboard.

Project Background

This project was initially based on a DataCamp DataLab code-along project, which provided the foundation for the exploratory analysis and data preparation.

I expanded the original project by adding my own analyses, visualizations, and interactive dashboard, including:

Czech Republic medal performance over time
Medal distribution by sport
Percentage share of medals by sport
Summer vs. Winter Olympics comparison
Czech Republic vs. Slovakia medal comparison
Additional Plotly visualizations
Interactive Streamlit dashboard

The purpose of these additions was to practice independent data analysis beyond the original guided exercise.
